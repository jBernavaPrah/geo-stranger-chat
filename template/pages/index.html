Welcome to Uploaderjs.com

Web site under construction.

Upload to your form from users smartphone!

With our simple script user can upload documents (photos, videos, pdf, ecc..) directly from their smartphone, without need to use other systems and without the need to install any apps.

Main features

  • Free service for you and for your users.
  • Simple configuration.
  • More conversions, improve usability and happynest of your users.
  • Secure: All files are encrypted before upload. Only you can decrypt its. (In development.)
  • All file are automatically deleted from our server after 24 hours or after your success download from your server.
  • Simple server side implementation in PHP, Python, Nodejs, outofbox. (See faq)
  • Customizable: max files, max file size, accept only mime types, ecc.. (See faq)

How can I use Uploaderjs?

Get it

Step 1: Create account:

Create account and configure it.

Step 2: Add script into form:

Add below script into forms tag.

{{ highlight(uploaderjs_script())|safe }}

Step 3: Request all files uploaded:

If user wish use this service, in your form will be added a new hidden input, with name "uploaderjs_request_id". With this id, you can perform a request to our api server. After you download the files, and your webserver complete the download, automatically will be deleted. All files will be saved for 7 days and then deleted.
